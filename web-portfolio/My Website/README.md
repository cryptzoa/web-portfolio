# Portfolio Website Rayhan

Website : https://kexterr-v1.netlify.app/

## Deskripsi

Website portfolio ini dibuat sebagai tugas kuliah untuk memperkenalkan diri sebagai mahasiswa Teknologi Informasi. Website ini menampilkan informasi pribadi, keterampilan, dan kontak saya sebagai programmer pemula. Desainnya sederhana, responsif, dan menggunakan animasi untuk memberikan pengalaman yang menarik.

## Fitur Utama

- **Halaman Utama (Home)**: Pengenalan diri dengan efek typewriter untuk jabatan "Programmer", tautan media sosial, dan tombol "Hire Me".
- **Halaman Tentang (About)**: Deskripsi singkat tentang minat dan kepribadian saya sebagai mahasiswa TI yang memiliki rasa ingin tahu tinggi, mudah beradaptasi, dan konsisten dalam pekerjaan.
- **Bagian Keterampilan (Skills)**: Menampilkan tingkat kemahiran dalam HTML (90%), CSS (80%), dan JavaScript (70%), serta pengalaman 0 tahun (karena masih mahasiswa).
- **Bagian Kontak (Contact)**: Ajakan untuk proyek dan tombol "Let's Chat".
- **Navigasi Responsif**: Menu hamburger untuk perangkat mobile dengan dropdown.
- **Animasi**: Menggunakan library AOS (Animate On Scroll) untuk efek zoom, fade, dan lainnya.
- **Footer**: Informasi hak cipta dan kredit pembuat.

## Teknologi yang Digunakan

- **HTML5**: Struktur dasar website.
- **CSS3**: Styling, responsivitas, dan animasi (termasuk media queries untuk mobile).
- **JavaScript**: Efek typewriter dan fungsi menu hamburger.
- **AOS (Animate On Scroll)**: Library untuk animasi saat scroll.
- **Font Awesome**: Ikon untuk media sosial dan navigasi.
- **Google Fonts (Poppins)**: Font utama untuk tampilan yang modern.

## Struktur File

- `index.html`: Halaman utama.
- `about.html`: Halaman tentang.
- `css/style.css`: File styling utama.
- `js/app.js`: Script JavaScript untuk interaktivitas.
- `assets/images/img.jpg`: Gambar profil.
- `README.md`: File ini.
